### 权重总览

| 维度 | 满分 | 超短线权重逻辑 |
|-----|-----|------------|
| 一、资金流向 | 30分 | 超短线本质是跟资金，主力净流入是最直接的买入信号，权重最高 |
| 二、分时走势 | 25分 | 超短线胜负往往在入场时机，分时决定买在哪个点位 |
| 三、板块共振 | 20分 | 超短线吃的是情绪和题材热度，板块轮动是核心逻辑 |
| 四、日K结构 | 10分 | 超短线只需确认趋势方向未破与动能强弱，避免逆势，权重不宜过高 |
| 五、风险排查 | 15分 | 含一票否决与中间风险扣分，权重提升以防止风险信号被强势维度掩盖 |
| 合计 | 100分 | |

### 维度一：资金流向（满分30分）

#### 子项1：今日主力净流入量级（满分12分）

主力 = 超大单 + 大单。用流通市值占比衡量（跨个股可比）：
`主力净流入占比 = 主力净流入额 ÷ 流通市值 × 100%`

| 占比区间 | 得分 |
|---------|-----|
| > +0.3% | 12 |
| +0.2% ~ +0.3% | 10 |
| +0.1% ~ +0.2% | 7 |
| +0.01% ~ +0.1% | 4 |
| -0.05% ~ +0.01% | 1 |
| < -0.05% | 0 |

#### 子项2：超大单 / 大单结构（满分10分）

| 超大单 | 大单 | 含义 | 得分 |
|------|-----|-----|-----|
| 正 | 正 | 机构+游资共振，最强信号 | 10 |
| 负 | 正 | 游资强势接力，可跟但需谨慎 | 6 |
| 正 | 负 | 机构建仓、游资分歧，中性偏多 | 4 |
| 负 | 负 | 主力全线撤退 | 0 |

#### 子项3：近5日主力净流入趋势（满分8分）

| 趋势特征 | 得分 |
|---------|-----|
| 5日累计为正，且近3日连续流入 | 8 |
| 5日累计为正，但有1-2日间歇流出 | 6 |
| 5日累计为负，但今日明显转正（占比 > 0.1%） | 4 |
| 5日累计为正，但今日转为净流出 | 2 |
| 5日累计为负，今日继续流出 | 0 |

### 维度二：分时走势（满分25分）

#### 子项1：开盘定性与全天趋势（满分8分）

开盘涨跌幅 = `（开盘价 - 昨收）/ 昨收 * 100%`

| 开盘 + 全天走势 | 得分 |
|--------------|-----|
| 平开/小高开（0-1%）后稳步走强，现价接近最高价 | 8 |
| 高开（>1%）后继续走强，无明显冲高回落 | 7 |
| 低开后快速拉升收复，整体走势偏强 | 7 |
| 平开后震荡，无明显方向但未大跌 | 4 |
| 高开后冲高回落，现价回落至开盘附近 | 2 |
| 开盘后持续走弱，现价接近最低价 | 0 |

#### 子项2：价格与均价线位置（满分9分）

均价线 = 当日各5分钟K线均价的加权均值（以成交量为权重）。
统计全天价格维持在均价线上方的时间比例：

| 价格与均价线关系 | 得分 |
|--------------|-----|
| >90% 时间在上方 | 9 |
| 60%-90% 时间在上方 | 7 |
| 40%-60% 时间在均价线附近反复震荡 | 4 |
| 价格跌破后被压制，偶尔触及均价线 | 1 |
| >70% 时间在均价线下方 | 0 |

#### 子项3：尾盘（14:30-15:00）表现（满分8分）

**半日数据处理**：若评估时间在14:30之前，尾盘子项标记为「待确认」，暂给中间分4分，收盘后需更新。

| 尾盘信号 | 得分 |
|---------|-----|
| 放量拉升，成交量 > 日内均量且价格创日内新高 | 8 |
| 平稳略涨，量正常 | 6 |
| 缩量横盘，方向不明 | 4 |
| 小幅回落，但未创日内新低 | 2 |
| 跳水放量，14:30后快速下跌 > 1% | 0 |

### 维度三：板块共振（满分20分）

#### 子项1：所属行业板块涨幅（满分6分）

取 `相关板块 → 行业` 中最强相关行业的涨跌幅：

| 行业涨幅 | 得分 |
|---------|-----|
| > 3% | 6 |
| 2% - 3% | 5 |
| 1% - 2% | 3 |
| 0.5% - 1% | 2 |
| 0% - 0.5% | 1 |
| < 0% | 0 |

#### 子项2：是否当日主线（满分7分）

| 情形 | 得分 |
|-----|-----|
| 所属行业 = 主线板块，且涨幅排名前3 | 7 |
| 所属行业 = 主线板块，且涨幅排名前5 | 5 |
| 非主线，所属行业综合涨幅大于 1% | 5 |
| 非主线，所属行业综合涨幅大于 0% | 2 |
| 其他 | 0 |

#### 子项3：概念共振计数（满分7分）

统计 `相关板块 → 概念` 中当日涨幅 > 0.5% 的概念占比（共振数 / 总概念数）。多概念共振是超短线获取超额溢价的关键，权重提升：

| 共振概念占比（涨幅>0.5%） | 得分 |
|----------|-----|
| ≥ 40% | 7 |
| 25% - 39% | 5 |
| 15% - 24% | 3 |
| 5% - 14% | 1 |
| < 5% | 0 |

### 维度五：风险排查（满分15分）

**⚠️ 一票否决项（触及任意一条 → 本维度0分，且禁止一切买入操作）**：

| 否决项 | 数据来源 | 备注 |
|-------|---------|-----|
| 实控人/大股东减持 > 流通市值1% | 快讯 | |
| 解禁规模 > 流通股5% | 快讯 | |
| "监管问询"、"异动警示"、"立案调查" | 快讯 | |
| 量比 > 5 且涨跌幅 < 0%（对倒嫌疑） | 量比+涨跌幅 | 涨跌幅<0%才否决，微涨(0-2%)不否决 |
| 换手率 > 20%（流通市值<50亿时>25%） | 换手率+流通市值 | 小盘股活跃度高，阈值放宽 |
| "业绩大幅低于预期"、"计提减值"、"业绩变脸" | 快讯 | |
| 北交所/科创板日成交额 < 5000万 | 成交额 | |

未触发一票否决，按以下3项打分：

#### 子项1：估值风险（满分5分）

取PE和PB分别评分，取两者中较低分作为本子项得分：

| PE区间 | 得分 | PB区间 | 得分 |
|-------|-----|-------|-----|
| 0 - 30 | 5 | 0 - 3 | 5 |
| 30 - 60 | 4 | 3 - 6 | 4 |
| 60 - 100 | 2 | 6 - 10 | 2 |
| > 100 | 1 | > 10 | 1 |

#### 子项2：技术异常（满分5分）

初始5分，触发以下每项扣分，最低0分：

| 异常信号 | 扣分 |
|---------|-----|
| 量比 2-5 且涨跌幅 < 1%（疑似对倒） | -1 |
| 振幅 > 5% 且收盘 < 最高价 - 振幅×0.5（长上影线） | -1 |
| 价格 > BOLL上轨 且量比 > 2 且涨幅 < 1%（高位量滞涨） | -2 |
| 近3日有2日以上出现长上影线 | -2 |
| 涨幅 > 3% 但量比 < 0.8（缩量上涨，动能衰减） | -2 |
| 连续2日放量但涨幅递减（价量背离） | -2 |

#### 子项3：新闻事件（满分5分）

| 新闻性质 | 得分 |
|---------|-----|
| 重大利多（业绩超预期/大额中标/增持回购/重大合作） | 5 |

| 轻微利空（小额减持/不分红/非核心高管离职） | 1 |
| 明显利空（业绩不及预期/诉讼纠纷/大额减持） | 0 |

### 维度冲突修正规则

评分完成后，检查维度间是否存在冲突，按以下规则修正总分。
**重要：快讯/新闻加分不作为总分的一部分直接累加，仅作为风险排查维度内子项3的评分依据。**

| 冲突类型 | 修正规则 |
|---------|---------|
| 资金强+技术超买 | 资金流向≥24分 且 RSI6>75 → 总分-5，标注「资金强但技术超买，需等回调」 |
| 资金强+板块弱 | 资金流向≥24分 且 板块共振≤5分 → 总分-3，标注「资金独立行情，板块无共振支撑」 |
| 资金弱+技术强 | 资金流向≤8分 且 日K结构≥8分 → 总分-3，标注「技术面好但资金不配合，假突破风险」 |
| 板块强+资金弱 | 板块共振≥15分 且 资金流向≤8分 → 总分-2，标注「板块情绪好但个股资金不跟，被动上涨」 |
| 分时强+风险异常 | 分时走势≥20分 且 风险排查≤5分（未触发否决） → 总分-3，标注「走势强但存在技术异常或利空，强势可能不可持续」 |
| 日K弱+资金强 | 日K结构≤2分 且 资金流向≥24分 → 总分-2，标注「资金流入但趋势未确认，需观察次日K线确认方向」 |
| **板块强+大盘弱** | 板块共振≥15分 且 大盘综合评分<5 → 总分-3，标注「板块独木难支，大盘拖累概率高」 |
| **RSI背离无放量** | RSI底背离（价格新低但RSI未新低）且量比<1.0 → 总分-3，标注「底背离无量能配合，假信号概率高」 |
| **尾盘偷袭+缩量** | 尾盘拉升>2% 且 全天量比<1.0 → 总分-4，标注「尾盘诱多嫌疑，次日高开低走概率大」 |
| **高估值+纯情绪** | PE>150 且 新闻事件得分≥4 → 总分-2，标注「高估值靠情绪支撑，消息落空后回调风险大」 |

### 快讯处理规则

- 快讯内容**仅用于风险排查维度→子项3：新闻事件（满分5分）的评分**
- **禁止**将快讯作为独立加分项直接加到总分上
- 重大利空快讯直接触发一票否决
- 重大利多快讯最多使新闻事件子项得5分（已包含在15分风险排查内）
